Inputs

Input Type Default Descripción
blockItems DdrBlockItem [ ] [ ] Items a mostrar.
labelNoResults string 'No results' Texto cuando no hay bloques.
showHeader boolean true Indica si queremos mostrar la cabecera.
showInfoAdditional boolean true Indica si queremos mostrar la información adicional.
showActions boolean true Indica si queremos mostrar las acciones.
showBorder boolean true Indica si queremos mostrar el borde personalizado.
pagination number 10 Indica el numero de bloques a mostrar.
previousLabel string 'Previous' Mensaje en el enlace de previo.
nextLabel string 'Next' Mensaje en el enlace de siguiente.

Outputs

Output Description
actionSelected Evento cuando se selecciona una acción.
itemSelected Evento cuando se selecciona un item.
