Inputs

Input Type Default Descripción
options DdrSelectItem[ ] [ ] Las opciones a mostrar.
valueSelect any null Indica el valor que se debe de precargar, si existe.
labelNoResults string 'No results' Muestra el texto cuando no hay opciones disponibles.

Outputs

Output Description
select Evento cuando se clicka en una opción.
